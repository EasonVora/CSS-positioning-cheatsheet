CSS Position Properties

Position Property Summary

In CSS, the position property is a tool that determines how and an element is placed on a webpage. By specifying the elements position in regards to other elements and/or the browser window, it helps to control the layout and arrangement of elements. Understanding how to implement the position property can imporve the ability to build visually attractive, organized web designs

Static Value

The static position specifies that an HTML element is set to the default position through CSS.

* {

position:

static;

}

It cannot be adjusted by using offset properties, but can interact with other positioning properties. Static elements appear in the order they are in on the HTML document. Additionally, static elements static elements will effect overlapping elements, and push down elements that are after it.

The Relative Value

The relative position uses offset properties to move its position relative to it's default position.

* {

position:

relative;

}

Without offset properties, the position is static by default. When offset properties are used, the element will move without affecting surrounding elements. Use the z-index to adjust the stacking order.

The Absolute Value

Absolute elements are ignored by other elements. It is positioned relative to it's nearest positioned ancestor, or alternatively to it's initial containing block.

* {

position:

absolute;

}

An absolute element can be adjusted with offset properties and by the z-index.

The Fixed Value

A fixed element will stay in the same place on the web page view port. It will stay fixed in place as the user scrolls throught the webpage.

* {

position:

fixed;

top: 0;

left: 5px;

}

Fixed elements and be also be adjusted using offset properties and by the z-index.

The Sticky Value

Sticky elements toggle between a fixed position and a relative position. The sticky elements stick to a specified position until the user scrolls past the defined scroll point using offset properties.

* {

position:

sticky;

top: 0;

left: 5px;

}

Sticky elements can also be adjusted using the z-index.

The Offsets Property

Offset properties in CSS are used to determine the position of elements that are set with, relative, absolute, fixed, or sticky positioning. They enable you to specify how far an element should be moved from its original location or from the edges of its containing block.

* {

position: relative;

top:

20px;

left:

30px;

}

The Main Four Offset properties Descriptions
Top (moves the element down away from it's original position)
Right (moves the element left away from it's original position)
Bottom (moves the element up away from it's original position)
Left (moves the element right away from it's original position)

The Z-Index Property

The z-index determines vertical stacking order of elements that overlap each other. Higher values appear in front of elements with lower numbers. The value for the z-index can be a negative number as well.

* {

position: relative;

z-index:

1000;

}
